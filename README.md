Display a [Le Wagon](http://www.lewagon.org) tickets status with [blink1](http://blink1.thingm.com/).

## Config

Open `config.yml` and edit the `batch_slug`.

## Launch

Start the deamon

```bash
# Clone the project and cd to it
$ ./deamon.rb start
```

Stop the deamon

```bash
$ ./deamon.rb stop
```
